🖥️ Comandos Windows

Referência Rápida de Utilitários e Configurações do Sistema

📁 Acesso a Pastas (Shell Commands)

Atalhos rápidos para acessar pastas importantes do sistema. Digite o comando na caixa Executar (Win + R).

shell:appsfolder
Pasta de Apps - Todos os aplicativos instalados
shell:startup
Pasta de inicialização (autoexec) - Aplicativos que iniciam automaticamente
shell:sendto
Pasta "Enviar para" - Opções de envio rápido
shell:downloads
Pasta Downloads - Arquivos baixados
shell:documents
Pasta Documentos
shell:pictures
Pasta Imagens
shell:desktop
Área de Trabalho

⚙️ Painéis de Controle e Utilitários

Acesso direto a configurações avançadas do sistema e painel de controle.

appwiz.cpl
Painel de Controle - Desinstalar/gerenciar programas
control
Painel de Controle geral - Menu principal de configurações
msconfig
Utilitário de Configuração do Sistema - Gerenciar inicialização
ncpa.cpl
Conexões de Rede - Configurar e diagnosticar redes
mmsys.cpl
Som e Dispositivos de Áudio - Configurar áudio
devmgmt.msc
Gerenciador de Dispositivos - Diagnóstico de periféricos (câmera, webcam, etc.)
dxdiag
Ferramenta de Diagnóstico DirectX - Drivers gráficos, áudio e rede

🎛️ Ferramentas de Monitoramento e Diagnóstico

Ferramentas para monitorar desempenho, recursos e diagnosticar problemas.

resmon
Monitor de Recursos - Versão intuitiva e simplificada (CPU, memória, disco, rede)
taskmgr
Gerenciador de Tarefas - Monitorar e encerrar processos
sndvol
Volume do Sistema - Controle de áudio
osk
Teclado Virtual - Entrada de texto sem teclado físico
displayfusion
Configurações de Monitor (se instalado) - Gerenciar múltiplos monitores

🌐 Rede e Acesso Remoto

Ferramentas para gerenciar conexões de rede e acesso remoto.

mstsc
RDP (Conexão de Área de Trabalho Remota) - Acessar outro computador remotamente
ncpa.cpl
Conexões de Rede - Configurar conexões de rede

🔧 Comandos de Sistema (CMD/PowerShell)

Comandos avançados para executar no Prompt de Comando ou PowerShell (execute como Administrador).

shutdown /s /t 60
Desligar em 60 segundos (substitua 60 pelo número de segundos desejado)
shutdown /r /t 60
Reiniciar em 60 segundos
shutdown /h
Hibernar o sistema
taskkill /IM "nomeapp.exe"
Encerrar processo específico (substitua "nomeapp.exe" pelo nome do processo)
taskkill /F /IM "nomeapp.exe"
Forçar encerramento de processo (adiciona /F para forçar)
ipconfig
Exibir configuração de rede e IP
ipconfig /flushdns
Limpar cache DNS (requer administrador)
chkdsk C: /F
Verificar integridade do disco C: (requer agendamento de reinicialização)

💡 Dicas Rápidas para Usar Esses Comandos

  • Para acessar Executar: Pressione Win + R (tecla do Windows + R)
  • Para abrir CMD: Pressione Win + X e selecione "Prompt de Comando (Admin)"
  • Para abrir PowerShell: Pressione Win + X e selecione "Windows PowerShell (Admin)"
  • Lembrete importante: Alguns comandos exigem privilégios de Administrador
  • Para copiar comandos: Selecione o texto, pressione Ctrl+C e cole com Ctrl+V
  • Modo seguro: Para diagnósticos, reinicie em Modo Seguro via msconfig
